* 调试基库版本1.7.2
* 根目录数据包data.json 需要挂载到服务器使用, 我使用的是集成服务器JspStudy，php版本只需要放到phproot目录引用就可以。同时修改项目的app.js文件，修改wx.request的url指向 http://你的IP地址/data.json
* 真机测试请使用你自己的内网IP，使用注册好的微信AppID，填写request合法域名，并且主机和手机在同一个局域网就可以测试。
